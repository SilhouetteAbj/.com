Build a premium healthcare diagnostic marketing website for Silhouette Diagnostics Consultants.

The platform will promote diagnostic services, generate patient leads, allow real-time communication with visitors, and provide an admin dashboard for managing bookings, chats, and analytics.

The site must be:

• modern
• interactive
• highly animated
• smooth and responsive
• mobile-first

The UI, motion design, and user interaction must closely match the behavior, feel, transitions, and animation style shown in the reference video provided by the client.

The final product should feel like a premium interactive medical platform, not a static website.

The website must:
